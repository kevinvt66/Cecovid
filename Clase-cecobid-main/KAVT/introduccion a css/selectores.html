Selectores de CSS

Los selectores tienen la función de encontrar o seleccionar elementos HTML a los que queremos dar estilo

  • Selectores sencillos - Apuntan a elementos basándose en su clase ., id # o nombre h1.
  • Combinadores - Selectores basados en una específica relación entre los elementos .contenedor div.
  • Pseudo-clases - Selectores basados en el estado del elemento button:hover.
  • Pseudo-elementos - Selecciona una parte específica de un elemento
  • De atributo - Selecciona elementos basándose en un atributo o en el valor de un atributo.

Selectores de nombre de elemento

Selecciona los elementos basándose en su nombre de elemento.

Ejemplo: Todos los span serán de color morado. 

                span {
                    color: purple;
                }

Selectores de id

Este selector usas el atributo id de un elemento HTML para seleccionar un elemento específico.

La id de un elemento es única en el documento. Por lo tanto, el selector de id # se usa para especificar un único elemento.

Ejemplo: Este párrafo será el único en estar centrado y de color verde limón.

Selectores de clase

El selector de clase . selecciona aquellos elementos con un atributo de clase específico.

Los selectores pueden especificar elementos dentro de un clase.

El selector universal

El selector universal * selecciona a todos los elementos.

Selector de agrupación

Sirve para agrupar aquellos selectores que tienen las mismas caraterísticas

Ejemplo: Digamos que queremos darle la misma sobra que .pSombras a todos los h1 y h2. En vez de ir a todos los elementos h1 y h2 y darles esta clase podemos hacer lo siguiente: 

        
            .pSombras {
                box-shadow: 5px 5px 3px -2px;
            }
            h1 {
                box-shadow: 5px 5px 3px -2px;
            }
            h2 {
                box-shadow: 5px 5px 3px -2px;
            }

Agrupado quedaría así:

        
            .pSombras, h1, h2 {
                box-shadow: 5px 5px 3px -2px;
            }
