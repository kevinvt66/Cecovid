Objetos en JS

Un objeto es un tipo especial de variable, que almacena distintos valores clasificados como propiedades.

Un objeto tambien puede almacenar distintos metodos estos son bloques de codigo que se ejecutaran en determinados momentos.

En la vida real, un coche es un objeto.

Un coche tiene propiedades como peso colos, y métodos como arrancar y parar.

Todos los coches tienen las mismas propiedades , péro éstas tienen un valor distintos dependiendo del coche

Todos los coches tienen los mismos métodos, pero éstos se pueden ejecutar en distintos momentos

Normalmente, usamos una variable para guardar un valor sencillo:

let code = 'Renault'

Los objetos son variables que pueden contener múltiples valores

    
            const coche={
            marca:"Renault",
            modelo:"Megane",
            color:"gris";
            }

Los valores se escriben emparejados con el nombre de la propiedad name:value
