El objeto fecha Date()

Nos permiten trabajar con las fechas:

El objeto Date() es estático. No es un reloj como tal, es un objeto cuyas propiedades se generan cuando el navegador carga el documento.

Creando Objetos Fecha

Los objetos de fecha se crean con new Date().

Hay distintas maneras de crear uno. Si no le indicamos parámetro, el objeto se genera con la fecha y hora actuales.

  • new Date()
  • new Date("string de fecha")
  • new Date(año, mes, día, horas, minutos, segundos, ms)
  • new Date(milisegundos)

new Date('string de fecha')

Crear un objeto a partir de una fecha concreta. En esta caso será el 7 de febrero del 99. Ref al js en línea 4

new Date(año, mes...)

La fecha se crea introduciendo números. Esto genera ciertas situaciones especiales. Ref en línea 11

Nota: JavaScript cuenta los meses de 0 - 11. Si se introduce un mes mayor que 11, JS seguirá contando los meses del siguiente año.

new Date(milisegundos)

JavaScript almacena las fecha como el número de ms que transcurrido desde el 1 de enero de 1970.

El tiempo cero (Zero Time) es el 1 de enero de 1970, 00:00:00 GTM 0.

Ahora el tiempo/fecha es: milisegundos transcurridos desde el tiempo cero.

Imprimir fechas

Escribir new Date() convertirá la fecha automáticamente a un string.

Un formato más facil de leer se obtiene al usar el método toDateString(). Ref al js línea 24.

Tomar datos de fecha (Get Methods)

  • getFullYear() - Retorna el año como un número de 4 dígitos YYYY
  • getMonth() - Retorna el mes como número 0 - 11
  • getDate() - Retorna el día del mes como número 1 - 31
  • getDay() - Retorna el día de la semana como número 0 - 6
  • getHours() - 0 - 23
  • getMinutes() - 0 - 59
  • getSeconds() - 0 - 59
  • getMilliseconds() - 0 - 999
  • getTime() - Retorna el tiempo o los ms desde el tiempo cero.
