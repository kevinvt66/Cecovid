Sintaxis de JavaScript

La sintaxis es la serie de reglas que definen cómo se construyen los programas. Ref al js línea 1.

Valores de JavaScript

Hay dos tipos de valores: Fijos y variables.

A los fijos se les llama literales(Literals).

A los variables se les llama Variables.

Valores fijos, Literales

Los dos tipos de Literales son los números y los strings, o cadenas de caracteres.

Los números se escriben con o sin decimales.
10.50
1001

Strings son texto, puede ir escrito entre comillas dobles o simples.
"Jorge González"
'Jorge González'

Variables

En programación las variables se usan para almacenar valores.

JavaScript se sirve de las keywords var, let, const para declarar variables.

El signo = se usa para asignar valores a las variables.

Operadores

JavaScript usa operadores aritméticos (+ - * /) para computar valores.

Expresiones

Las expresiones son una combinación de valores, variables y operadores que computan en un resultado o valor. Ref al js línea 11.

Comentarios

Los comentarios son líneas de código que son ignoradas. No se ejecutan.

Tienen principalmente dos usos. El primero es explicar el código y el segundo es para desactivar cierta parte de nuestro código sin necesidad de borrarla.

En JavaScript hay dos maneras de comentar. En línea con // y en bloque con /* ... */

Identificadores/Nombres

Los identificadores se usan para nombras variables y funciones.

Hay una serie de reglas para nombrar comunes en la mayoría de lenguajes de programación.

Un identificador debe empezar por:

  • Una letra (A-Z o a-z)
  • Un símbolo de dólar $
  • o una barra baja _

Los caracteres siguientes pueden ser números, letras, _, $.

No se permite empezar con números!

Uso de mayúsculas

Los identificadores son sensibles a las mayúsculas. Esto quiere decir que lastName y lastname son dos variables diferentes.

Y, por ejemplo, JavaScript no interpreta LET o Let como la palabra clave let.

Camel Case

Antes, los programadores usaban guiones - para unir distintas palabras en un identificador. last-name, first-name...

Sin embargo, en JavaScript los guiones están reservados para las restas. Por lo que los programadores de JS se inclinan a separar las distintas palabras mediante mayúsculas, dejando el resto en minúsculas.

primerApellido, colorVehiculo, codigoPostal.

Como se ha podido apreciar, no es conveniente el uso de tildes, ñ u otros caracteres ajenos al inglés, idioma en el que se diseñado la programación.
