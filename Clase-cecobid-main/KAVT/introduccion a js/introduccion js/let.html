Todo sobre let

La palabra clave o keyword let tiene 3 usos principales.

  • Las variables declaradas con let no pueden ser redeclaradas.
  • Las variables definidas con let deben declararse antes de ser usadas.
  • Las variables declaradas con let tienen un alcance (scope) de bloque.

No se pueden redeclarar

En JavaScript, al usar var se puede, accidentalmente, redeclarar y cambiar el valor de un variable.

var x = 'Juan';
var x = 0;

No sólo perdemos el valor anterior, si no que esto puede generar errores.

let impide redeclarar variables y por tanto, es más seguro.

Alcance de bloque (block scope)

Las variables declaradas con let tienen alcance de bloque. Es decir, aquellas variables declaradas dentro de un bloque {} no pueden ser usadas fuera del bloque. Ref al js línea 1

Variable declaradas con let deben ser declaradas ante de usarse (hoisting)

Usando var podemos usar variables mientras éstas estén declaradas en cualquier parte del código.

Con let la declaración debe ser anterior al uso u obtendremos un error de referencia. Ref al js línea 29.
