Conceptos de JS

Un programa informático es una lista de instrucciones que se ejecutan por un ordenador.

En programación estas instrucciones se llaman declaraciones o "statements".

Un programa de JavaScript es una lista de declaraciones de programación.

Statements o declaraciones

Las declaraciones se componen de:

Valores(values), operadores(operators), palabras claves(keywords), y comentarios(comments).

Ejemplo: Tenemos una declaración(statement) que le indica al navegador que debe escribir "Hola Mundo" dentro de un elemento HTML con la id="test"

document.getElementById('test').innerHTML = 'Hola Mundo';

Los statements, o líneas de código, se ejecutan una a una en orden de lectura.

Punto y coma (semicolon) ;

Se usan para separar los statements en JS. No son obligatorios, pero sí muy recomendados, se usan por buenas prácticas de código. Ref al js línea 1.

Espacios en blanco

JavaScript ignora los espacios en blanco, y se usan para hacer el código más legible. Ref al js línea 11.

let persona = "Carlos";
let persona="Carlos";

Bloques de código

Las declaraciones o statements se pueden agrupar en bloques, dentro de llaves(curly brackets) {...}

El propósito de ésto, es definir una serie de declaraciones para que se ejecuten juntas.

Un ejemplo de declaraciones unidas en bloques son las funciones. Ref al js línea 16.

Palabras clave o keywords

A menudo, muchas líneas de código empiezan por un palabra que indica la acción a realizarse.

Estas palabras están reservadas y no pueden usarse en ningún otro contexto. Algunos ejemplos:

  • var - Declara una variable
  • let - Declara una variable de bloque
  • const - Declara una constante de bloque
  • if - Indica un bloque de código a ejecutarse cuando se cumple una condición
  • switch - Indica un bloque de código a ejecutarse dependiendo de distintos factores
  • for - Indica un bloque de código a ejecutarse en bucle
  • function - Declara una función
  • return - Sale de una función
  • try - Implementa manejo de errores a un bloque de código
