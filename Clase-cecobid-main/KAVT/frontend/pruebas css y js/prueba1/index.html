Animaciones con CSS

CSS permite animar elementos HTMLsin necesidad de recurrir a JS

Qué son las animaciones¿?

Una animación es el cambio gradual de un estilo a otro.
Se pueden cambiar tantas propiedades de CSS como se quiera, tantas veces como sea necesario.
Para usar una animación debe especificarse la regla @keyframes

La regla @keyframes

Esta regla contiene los estilos que el elementotendrá en determinados momentos.
Cuando se especifica una regla, los cambios se ejecutarán de manera gradual en ciertos momentos.
Para que la animación funcione, debes asignar un kewframe al CSS de un elemento.
