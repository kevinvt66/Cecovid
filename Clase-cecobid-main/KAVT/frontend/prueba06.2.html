Fecha y hora en JS

El objeto fecha Date()

Nos permite trabajar con las fechas:

El objeto Date() es estático. No es un reloj como tal, es un objeto cuya propiedades se generan cuando el navegador carga el documento.

Creando Objetos Fecha

Los objetos fecha se cran con new Date().

Hay distintas maneras de crear uno. Si no le indicamos párametro, el objeto se genera con la fecha y hora actuales.

  • new Date()
  • new Date('string de fecha')
  • new DAte('año,mes,dia,horas,minutos,segundos,ms')
  • new Date('milisegundos')

new Date('string de fecha')

Crear un objeto a partir de una fecha concreta. En este caso sera e 7 de febrero del 99. ref al js linea 4

new Date('año, mes, ...')

La fecha se crea introduciendo números. Esto genera ciertas situaciones especiales. Ref en linea 8.

Nota: JS cuenta los meses de 0-11. Si se introduce un mes mayor que 11, Js seguira contando los meses del siguiemte año.

new Date('milisegundos')

JS almacena las fechas como el número de ms que ha transcurrido desde el 1 de enero de 1970.

El tiempo cero(Zero time) es el 1 de enero de 1970, 00:00:00 GTH 0

Ahora el tiempo/fecha es: milisegundos transcurridos desde el tiempo cero.

Tomar datos de fecha (Get Methods)

  • getFullYear() - Retorno el año como un número de 4 digitos YYYY
  • getMonth - Retorna e mes como número 0-11
  • getDate() - Retorna el día de mes como número 1-31
  • getDay() - Retorna el dia de la semana como número 0-6
  • getHours() - 0-24
  • getMinutes() - 0-59
  • getSeconds() - 0-59
  • getAllSecnds() - 0-999
  • getTime() - Retorna el tiempo o los ms desde el tiempo cero
