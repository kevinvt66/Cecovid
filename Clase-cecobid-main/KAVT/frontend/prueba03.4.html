Listas HTML

Listas sin orden

Se usa la etiqueta ul como contenedor. Cada elemento de la lista irá en una etiqueta <li>:

Se usan para mostrar listas en las que el orden de los elementos no es importante. Como por ejemplo la lista de la compra, asistencia, material escolar, etc.

  • Fruta
  • Carne
  • Pasta
  • Leche

Listas ordenadas

Se usa la etiqueta <ol>(ordered list). Cada elemento de la lista irá en una etiqueta <li>.

Son listas numeradas. Se usancuando el orden es importante y los elementos tienen importancia según el órden o son pasos a seguir, por ejemplo.

Algunos ejemplos de listas ordenadas son una receta, un manual/libro que va ordenado por capítulos (la lista sería el índice).

Receta de café con leche

      Preparar el café
      Añadir la leche caliente
      Añadir azúcar al gusto
      Servir

Listas descriptivas

Son listas de términos, cada uno de éstos con su propia descripción. Se usa la etiqueta <dl>(descripción list) como contenedor, cada término irá en un ⌞dt> y cada descripción en un <dd>.
