Atributos de elementos HTML

Los atributos proveen información adicional de elementos HTML.

  • Todos los elementos HTML pueden tener atributos.
  • Los atributos proveen información adicional de los elementos
  • Los atributos se especifican en la etiqueta apertura
  • Los atributos normalmente tienen estructura de nombre="valor"

Ejemplo. Un link se define se defiende con la etiqueta < a >, El atributo "href" especifica la URL de la página a la que irá el link.
